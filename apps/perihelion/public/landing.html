Image Browser

Perihelion

Perihelion is an image/archive browser. It is built for my large personal collection, with staging, export/download/sharing, inspecting metadata, and managing tags/lists. Peri is a React 19 + Vite app with Tailwind CSS, lucide-react icons, and Motion, with a repo containing a TypeScript/Express local development server and live media python API.

The Python backend is located on a home server and serves media from a share location. It exposes auth status and admin/history endpoints, and uses Pillow when available for richer image metadata. It stores operational data in SQLite for local users, sessions, download history, and image details.

I wanted access to my image collection away from the desk so I could draw or paint with references. I wanted to move through pages easily, come back to specific shares without having to remember every one. This evolved into APIs and Cloudflare, metadata, large and non-image file handling and learning how to manage a home server. Securing Peri became the need for Auth/Multi, and organizing the files became the need for Vermilion. And it has the same file processing/download functions as Bullion, except items must be staged first.
