Vermilion

The current Vermilion is a React 19 + TypeScript SPA with a Python 3 HTTP API that splits folders with large amounts of files into smaller folders. The original Vermilion was a local utility using Python/tkinter, scanning a source directory, separating images from non-images, generating a destination tree, then executing. Spec sheet.

I created Verm for better folder management. It was originally just a local script with gui, but after creating the Millionfold web interface, I wanted to convert this into something that can be run on the server from whatever computer I am using. Like Peri and Mill (and Bullion), it accomplishes the purpose of handling large amounts of image files.
