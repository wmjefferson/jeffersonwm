Image Processor

Millionfold

Millionfold a web-based image processor using a React 19 + TypeScript SPA using Vite, and a Python 3 HTTP API. I have thousands of drawings on Google Keep Notes that vary in dimensions, and needed to equalize position and size for use in print and web. Manually this process has taken forever. With Millionfold I get every image, regardless of content, processed with a standard background (or transparency) in my desired format. All I have to do is collect the images.
